In a heavy contact sport like American football, injuries to athletes are very common, and when you consider the fact that an NFL roster consists of 53 players the opportunity for injuries becomes even more common. So how do teams deal with injuries? Do injuries make a significant impact on team performance? In this tutorial, I will be using the entire data science pipeline to provide some insights for answering these questions.

Data selection¶

Before invesitgating the impact of injuries in the NFL, I need to first gather some data about player injuries in the NFL. Unfortnunately, there was no database or dataset of player injuries that I could get my hands on, so I decided to scrape websites for data.

Pro-football reference is a catalog of statistics regarding NFL teams. The website also contains information regarding injuries. More specifically, the website has a page dedicated for injuries of a team over the course of a season. Each page contains a table which reports which players were reported as injuried, the row entries are players on the roster, and the columns is week opponent. Each player on a roster is reported as either IR, OUT, QUESTIONABLE, DOUBTFUL, (c19 for 2020) or None if not injuried and each are defined as.

Defintions of column names¶

IR- the player is on the injured reserve list
OUT - the player is out
DOUBTFUL - the player has a 25% chance of playing
QUESTIONABLE - the player has a 50% chance of playing
C19 - (only for 2020) the player on covid reserve list
Definition of the rest of initial columns after data collection:
record - team win loss ratio
expected_record - an estimate of what the team's record "should have been"
total_aval_players - number of players on roster * number of weeks played

Using python I then scrapped the injury data from each page, collecting the injury history of every team for the past 10 years. In addition, to injury history I also captured the record and expected record of each team for 10 years. According to pro-football reference the expected record is "an estimate of what the team's record "should have been," given the team's points scored and allowed." These to metrics will help us quantify a team's performance.

import requests
from bs4 import BeautifulSoup
import re
import pandas as pd

# league years and teams to look-up index for url
years = [2010 + i for i in range(11)] 
teams = ['buf','nwe','mia','nyj','clt','oti','htx','jax','pit','cle','rav','cin','kan','rai','den','sdg',
         'was','nyg','phi','dal','nor','tam','atl','car','gnb','chi','min','det','ram','sea','crd','sfo']

data = []

# making a get request for each team in each league year
for year in years:
    for team in teams:
        
        #the url's with injury and performance data
        url = f"https://www.pro-football-reference.com/teams/{team}/{year}_injuries.htm"
        url2 = f"https://www.pro-football-reference.com/teams/{team}/{year}.htm"
        
        #making a get request
        r = requests.get(url,headers={"User-Agent":"Chrome/84.0"})
        r2 = requests.get(url2,headers={"User-Agent":"Chrome/84.0"})
        
        #parsing the html response from server
        root = BeautifulSoup(r.content,"html")
        root2 = BeautifulSoup(r2.content,"html")
        
        # converting the html to a string to use regular expression to extract
        # record, play-off appearance, and expected record
        s = str(root2) 
        pattern = r"\<strong\>Record:\<\/strong\> ([0-9]{1,2}-[0-9]{1,2}-[0-9])"
        record = re.search(pattern,s).group(1)
        pattern = r"Expected W-L\<\/a\>:\<\/strong\> ([0-9]{1,2}.[0-9]{1,2}-[0-9]{1,2}.[0-9]{1,2})"
        expected_record = re.search(pattern,s).group(1)
        pattern = r"\<strong\>Playoffs:\<\/strong\>"
        playoff = re.search(pattern,s)
        playoffs = 1 if playoff else 0
        
        #finding the table from the other get request and converting to a dataframe 
        table = root.find('table')
        df = pd.read_html(str(table))[0]
        
        total_aval_players = len(df.columns)*len(df)
        
        #iterating through dataframe and counting the number of injuries for the year
        OUT = 0
        IR = 0
        C19 = 0
        doubtful = 0
        for index,row in df.iterrows():
            for week in df.columns:
                if row[week] == 'IR':
                    IR = IR + 1
                elif row[week] == 'O':
                    OUT = OUT + 1
                elif row[week] == 'D':
                    doubtful = doubtful + 1
                elif row[week] == 'C19':
                    C19 = C19 + 1
        
        #adding obervation to that dataset
        row = [team,year,OUT,IR,doubtful,C19,(OUT+IR+C19+doubtful),total_aval_players,record,expected_record,playoffs]
        data.append(row)

# making a dataframe with all the scrapped data and labeling columns
col = ['team','year','out','IR','doubtful','C19','total','total_aval_players','record','expected_record','playoffs']
injury_data = pd.DataFrame(data,columns=col)

Data processing¶

After collecting the raw data, we want to process it to gain more insight. First we want to convert the record and expected record to winning percentage and expected winning percent, to normalize team performance across league years. In addition, we also want to compute the ratio of the amount of games each player didn't play due to injury with the amount of games each player could have played if healthy.

In [260]:
dt = cpy.copy()

winning_percentage = []
expected_winning_percentage = []
percent_of_players_lost_to_injury = []

#ierating through each observation and computing new values
for index,row in dt.iterrows():
    
    #using regular expression to extract record and winning record
    q = re.search(r"([0-9]{1,2})-([0-9]{1,2})-([0-9]{1,2})",row['record'])
    q2 = re.search(r"([0-9]{1,2}.[0-9]{1,2})-([0-9]{1,2}.[0-9]{1,2})",row['expected_record'])
    w = int(q.group(1))
    ew = float(q2.group(1))
    l = int(q.group(2))
    el = float(q2.group(2))
    d = int(q.group(3))
    
    #calculating percentage
    winning_percentage.append(((w+(d/2))/(w+l+d)))
    expected_winning_percentage.append(ew/(ew+el))
    percent_of_players_lost_to_injury.append(row['total']/row['total_aval_players'])

# adding new columns and removing unused ones
dt['winning_percentage'] = winning_percentage
dt['expected_winning_percentage'] = expected_winning_percentage
dt['percent_of_players_lost_to_injury'] = percent_of_players_lost_to_injury
dt = dt.drop(['total','record','C19','out','IR','doubtful','total_aval_players','expected_record'],axis=1)

Model selection¶

We will fit a model on a teams using bunch of binary features, and they are winning_record? and expected_winning_record? (question mark indicates it a 1 = yes or 0 = no question), and also the injury record difference from the league average for each year.

We chose to use knn and decision tree as our initial classifiers and ran 10 fold cross validation to pick which model fit better with our data. After our tests knn had an accurary 98% so we chose to use knn to make some classifications to see if a team would be considered to be impacted by injuries.more details below...

from sklearn.tree import DecisionTreeClassifier
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score
from sklearn.model_selection import KFold
import numpy as np


X = dt[['expected_winning_record','winning_record','inj_diff']]
y = dt[['exp_imp']]

def Decision_Tree_TenFoldCV():
    #shuffling and spliting dataset into 10 bins
    d = KFold(10,shuffle=True)
    score = []
    for train_index,test_index in d.split(X):
        # splitting dataset into training and testing batches
        tr_data_x = pd.DataFrame([ X.iloc[i] for i in train_index])
        te_data_x = pd.DataFrame([ X.iloc[i] for i in test_index])
        
        tr_data_y = pd.DataFrame([y.iloc[i] for i in train_index])
        te_data_y = pd.DataFrame([y.iloc[i] for i in test_index])
        
        # training the training dataset with a decision tree
        m = DecisionTreeClassifier(max_depth=4).fit(tr_data_x,tr_data_y)
        
        # making a predicton with testing dataset
        pr_data_y = m.predict(te_data_x)
        
        # scoring the accuracy of the prediction
        score.append(accuracy_score(pr_data_y,te_data_y))
    return np.asarray(score)

def knn_TenFoldCV():
    #shuffling and spliting dataset into 10 bins
    d = KFold(10,shuffle=True)
    score = []
    for train_index,test_index in d.split(X):
        # splitting dataset into training and testing batches
        tr_data_x = pd.DataFrame([ X.iloc[i] for i in train_index])
        te_data_x = pd.DataFrame([ X.iloc[i] for i in test_index])
        
        tr_data_y = pd.DataFrame([y.iloc[i] for i in train_index])
        te_data_y = pd.DataFrame([y.iloc[i] for i in test_index])
        
        # training the training dataset with knn
        m = KNeighborsClassifier(n_neighbors = 5).fit(tr_data_x,np.ravel(tr_data_y,order='C'))
        
        # making a predicton with testing dataset
        pr_data_y = m.predict(te_data_x)
        
        # scoring the accuracy of the prediction
        score.append(accuracy_score(pr_data_y,te_data_y))
    return np.asarray(score)
